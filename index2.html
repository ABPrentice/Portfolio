Knock Alert

Knock Alert is a system that is composed of a desktop application and an embedded device. The system is designed to detect door knocks and alert the user to them. I was in charge of sending messages over Bluetooth from the embedded device to the desktop computer.

Next Gen

Next Gen is an open world Roblox game that allows players to collect and build in a futuristic city while surviving and dealing with obstacles. I was responsible for creating the UI and inventory system while also modelling the in-game items. The game is currently unreleased.
